## 📌 Project Overview

A Natural Language Processing (NLP) based ML project that detects stress and emotions from text, and provides calming advice, breathing exercises, and communication tips to help family members de-escalate conflicts and improve understanding between each other.

**Anyone can type how they feel — in simple words — and get helpful, kind guidance.**


## 🤖 What It Does

1. Person types how they feel (e.g. "I am very angry", "I feel sad and ignored")
2. ML model detects the **emotion** (Angry / Sad / Anxious / Frustrated / Calm)
3. System gives:
   - 💨 **Breathing exercise** to calm down
   - 💬 **Kind words** to feel understood
   - 🗣️ **Communication tip** to talk better with family
   - 🧘 **Activity suggestion** to divert the feeling
